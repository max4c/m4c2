I enjoy tools that are pieces of art in and of themselves. This page is inspired by Kevin Kelly's [CoolTools](https://kk.org/cooltools/) site. Below is a continually updated list of tools I use in my personal and professional life that spark joy.

#### Everyday Productivity

- [Raycast](https://www.raycast.com) - I use it ~100 times a day for so many things. It is the foundation of my efficiency on macOS.
- [Shottr](https://shottr.cc) - So many useful tools baked into one: ocr, scrolling screenshots, quick edits, etc.
- [Linear Mouse](https://linearmouse.app) - Increase cursor movement than highest speed on Macos. My pointer speed is 0.13, do not go to the max speed as it will be extremely hard to reset it!
- [Notion Calendar](https://www.notion.so/product/calendar) - Good design. I preferred when it was named Cron
- [Ice](https://icemenubar.app) - Hide apps in the menu bar. Clean, modern design and actively maintained.
- [Bitwarden](https://bitwarden.com) - Password autofill is so satifying. Simple and it just works.
- [Superwhisper](https://superwhisper.com) - To send longer prompts to LLMs quickly, speech to text is so satisfying and this app is infinitely better than the normal dictation on macos and runs locally AKA free. 
- [Karabiner-Elements](https://karabiner-elements.pqrs.org) - Keyboard customizer. I mapped my FN key to trigger Superwhisper so I can talk more and rarely touch the keyboard.

#### Terminal

- [Ghostty](https://ghostty.org) - So clean. The default place for launching agents for now.
- [Cmux](https://cmux.com) - On the verge of replacing Ghostty for me. Notifications when an agent needs me, vertical tabs, and split panes. Just waiting for it to get a little less buggy.
- [Oh My Zsh](https://ohmyz.sh) - I love [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions) purely for spinning up Claude faster. It ghost-completes `--dangerously-skip-permissions` so I don't have to type it out.
- [Powerlevel10k](https://github.com/romkatv/powerlevel10k)
- [Zoxide](https://github.com/ajeetdsouza/zoxide) - A smarter cd. I love flying around the terminal with it. `z` to jump anywhere, `zi` for a fuzzy finder.
- [Eza](https://github.com/eza-community/eza) - A modern replacement for `ls` with colors, icons, and git integration.
- [bat](https://github.com/sharkdp/bat) - A `cat` replacement with syntax highlighting. Aliased `cat` to it.
- [fzf](https://github.com/junegunn/fzf) - Fuzzy finder for the terminal. Powers fuzzy history, file picking, and zoxide's `zi`.
- [Neovim](https://neovim.io) - For quick edits without leaving the shell.

#### Developer Tools

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) and [Codex](https://openai.com/index/introducing-codex/) - I go between both.
- [uv](https://github.com/astral-sh/uv) - Python package and project manager written in Rust. Absurdly fast, and it's replaced pip/venv/pyenv for me entirely.
- [Disk Inventory X](http://www.derlien.com) - best way to visually manage storage on macos.

#### AI Tools

- [gws cli](https://github.com/googleworkspace/cli) - So nice to have one CLI to do everything Google related. It's how I read and manage almost all my emails.
- [Paper](https://paper.design) - As an engineer learning to become a designer, it's nice to have it use a canvas.
- [Granola](https://granola.ai) - Meeting notes. I'm very close to replacing it with my own tool, but for now it's just so easy to use.

#### Notetaking

- [Notion](https://notion.so) - Has been my LifeOS for six years, but Claude Code is slowly erasing every need I've had for it. I'm leaning towards everything being file over app.
- [Obsidian](https://obsidian.md) - Local notetaking that makes a beautiful writing experience.
- [Pilot G-2 05](https://www.amazon.com/PILOT-G2-Pens-Refillable-Retractable/dp/B09GKY4QT1) - I handwrite in my journal nightly and this pen makes it so satifying.

#### Clothing

- [Ibex](https://ibex.com) - Sustainable lululemon that aligns with my interest in hiking and plastic-free clothing.
- [Ryker](https://rykerclothingco.com/) - plastic-free gym shorts were so incredibly hard to find. Now I use these for running, hiking, lifting, and even swimming.

#### Health

- [Colemak](https://en.wikipedia.org/wiki/Colemak) - I never learned to touchtype until college when my wrists started to hurt a lot due to ignorance concerning propr ergonomics. I chose colemak as my keyboard layout and enjoy it so much. Instead of looking at a diagram of where letters are, I just started completely fresh on monkeytype and looked at the screen and let muscle memory develop over time. After 2 years, I am floating around 84 wpm with 96% accuracy
